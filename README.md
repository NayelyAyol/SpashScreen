<h1 align="center">Galería de Fotos con Splash Screen e Ícono Personalizado</h1>

---

## Descripción
- Aplicación móvil desarrollada en Ionic que permite tomar fotografías y visualizarlas dentro de la galería del dispositivo.  
- Incluye un ícono personalizado de la aplicación y un Splash Screen que se muestra al iniciar la app.

### Autora
- Nayely Ayol
---

## 📄 Características
- Captura de fotografías desde la cámara  
- Visualización de imágenes en la galería del teléfono y dentro de la app  
- Ícono personalizado de la aplicación  
- Splash Screen al iniciar la aplicación  
- Interfaz sencilla 

---

## 🛠️ Tecnologías utilizadas
- Android Studio  
- Ionic Framework  
- Visual Studio Code  

---

## 🚀 Instalación y ejecución

### APK
1. Descargar el archivo `.apk` del repositorio  
2. Instalarlo en un dispositivo Android  
3. Abrir la aplicación
